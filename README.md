## Project Overview
One-Material is a fictional advanced-materials company developing the WH-AGdTiB™ Al6061-Gd₂O₃-TiB₂ additively manufactured thermal-neutron absorber panel for spent-fuel storage baskets.

**Core message:** One-Material이 원전 안전의 미래를 설계합니다.

## Actual AI Production Workflow
Planning and prompt engineering → ChatGPT image generation → Kling AI image-to-video (Scenes 1-2) → Suno AI background music → ElevenLabs narration → CapCut final editing.

Because Kling credits were exhausted, Scenes 3-5 used AI-generated images with limited CapCut motion, text, and integration editing. This was the project’s documented fallback strategy under tool-access and credit constraints.

## Tools
| Tool | Actual use |
|---|---|
| ChatGPT | Planning, prompt engineering, five scene images |
| Kling AI | Image-to-video for Scenes 1-2 |
| Suno AI | AI-generated instrumental background music |
