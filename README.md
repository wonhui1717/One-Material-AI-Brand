# One-Material AI Brand Advertisement

## Project Overview

This project presents an AI-generated brand advertisement for **One-Material**, a fictional advanced materials company specializing in thermal neutron absorber panels for spent nuclear fuel storage baskets.

The advertisement was created entirely using generative AI tools, following the workflow of planning, prompt engineering, image generation, video generation, audio generation, and final editing.

# Brand Identity

One-Material develops next-generation thermal neutron absorber panels for spent nuclear fuel storage baskets.

The company focuses on three core values:

- Cost-effective manufacturing
- High thermal neutron absorption performance
- Simple manufacturing process

# Campaign Goal

Build a premium engineering brand image while introducing an innovative neutron absorber panel technology for safer nuclear power plants.
